# court-bot
Discord integration with objection.lol/courtroom for two-way messaging

## Projects

### CourtBot (Main Project)
Discord bot that bridges Discord channels with objection.lol courtroom chat.

**Files:** `courtbot.py`, `docker-compose.yml`, `Dockerfile`
**Documentation:** See `.github/copilot-instructions.md`

### Music Scraper
Scrapes all music tracks from objection.lol API for database creation.

**Location:** `music-scraper/`
**Documentation:** See `music-scraper/README.md`

### Music Uploader
Upload music from YouTube to objection.lol in one command.

**Location:** `music-uploader/`
**Documentation:** See `music-uploader/README.md`
